OpenSpec é um CLI e workflow open-source para **spec-driven development** que faz a ponte entre humanos e assistentes de IA: cria/organiza especificações, propostas e tarefas, valida deltas de requisitos e mantém um histórico reviewable antes de gerar/alterar código.

## Resumo das etapas:

**Inicializar** openspec init -> Cria a pasta openspec/
**_1- Preencha o project.md com o contexto(detalhes) do seu projeto_** project.md (Contexto do domínio, Pilha de tecnologias, Convenções de código, Padrões arquiteturais)
**_2- Solicite a funcionalidade(proposta)_** (gera e valide os arquivos: spec, design,proposal)
**_3- Implemente a funcionalidade(proposta)_** (gera e valide os arquivos: spec, design,proposal)
**_4- Arquivas as mudanças_** Marca a mudança como concluída
